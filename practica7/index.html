Conexión Bluetooth

Andrea Tapia Balvanera
Ingeniería Mecatrónica
Octubre 18, 2024
Arduino Electrónica Bluetooth

- Resumen -

En esta práctica utilizamos MIT App Inventor para programar una aplicación móvil donde podíamos controlar un circuito conectándolo mediante Bluetooth a otro dispositivo, donde podiamos encender y apagar la pantalla y un LED.

- Introducción -

MIT App Inventor es una plataforma de desarrollo visual que permite crear aplicaciones móviles de manera sencilla mediante bloques de programación.

Está diseñado para facilitar el desarrollo de apps para dispositivos Android, permitiendo a los usuarios controlar dispositivos externos, como sensores o actuadores, a través de conexiones Bluetooth.

El objetivo de esta práctica fue crear una aplicación que controlara un LED a través de una conexión Bluetooth con un Arduino, lo que nos permitió entender cómo integrar la programación móvil con la electrónica para el control remoto de dispositivos físicos.

- Desarrollo -

Para la parte de encender y apagar la pantalla, primero diseñamos la interfaz de la aplicación, donde creamos dos botones, uno para encender y otro para apagar la pantalla.

Programamos mediante bloques de código para que, al presionar el botón de encender, la pantalla se pusiera en blanco, y al presionar el botón de apagado, cambiara el color a negro./p>

Para encender y apagar un led, diseñamos la interfaz colocando un botón para conectarse mediante Bluetooth, otro para desconectarse y otro para salir de la aplicación. Además, creamos dos botones para encender y apagar el LED ya estando conectado a un dispositivo.

Para programarlo, utilizamos bloques de código donde colocamos las diferentes acciones que realizaba cada botón y configurarlo para la conexión Bluetooth.

Diseño

Para la parte del diseño, utilizamos MIT App Inventor para crear la interfaz de la aplicación.

Creamos diferentes botones para cada función y que sea más fácil para el usuario entender para que sirve cada uno.

Construcción

La construcción del programa fue mediante diferentes bloques de código, donde los programamos para que realicen diferentes funciones, así como enviar señales para encender y apagar un LED, y otros para conectar y desconectarlo de un dispositivo por Bluetooth.

Después, para poder encender un LED, creamos un código en Arduino que nos permitió conectarlo por Bluetooth y construimos el circuito.
